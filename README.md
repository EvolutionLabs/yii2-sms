# yii2-sms

This is a component used to send sms, mimic the yii mailer component behaviour


#### Installation

via composer.json

```json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "git@gitlab.cartu.ro:components/yii-sms.git"
        }
    ]
}
```

```shell script
composer require evolutionlabs/yii2-sms
```
